# Background images

Drop the interview room background here using this exact filename. The app
references it at `/backgrounds/room.png`.

| File       | Used on                                                        |
| ---------- | ------------------------------------------------------------- |
| `room.png` | Behind the avatar on all interview screens (`/interview...`) |

Until this file is added, the interview stage falls back to a solid purple
background. Once `room.png` exists at this path it appears automatically.
